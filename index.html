My projects

An abridged list of my publicly available projects.

LuckPerms

  • LuckPerms is a permission plugin for Minecraft servers, written in Java.
  • It has (at time of writing) ~200,000 monthly active users, and a Discord community of ~8000 people.
  • More information can be found on the project homepage, luckperms.net.

spark

  • spark is a performance monitoring plugin with a built-in profiler for Minecraft servers, written in Java and HTML/JavaScript.
  • More information can be found on the spark and spark-web GitHub pages.

helper

  • helper is a Bukkit plugin development toolkit. Designed for plugin developers, it provides a number of utilities and extended APIs to support rapid and easy development of plugins.
  • More information can be found on the helper GitHub page.
